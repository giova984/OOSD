Detailed Description

template<class X>
class MetaSim::GEvent< X >

This template class is used for simplifying the writing of a system model. Most of the times, the doit() method of an event must only call a handler method of some class. Normally, you must write a new class derived from event and override the virtual doit() method. However, it is a lot of repetitive work! And when there is a lot of repetitive work to do, template is the solution!

The GEvent class takes a class derived from Entity as a template argument. Then, template function register_handler() must be called passing the pointer to the handler before using this event.
